# -Math-Treasure-Hunt
🏴‍☠️ Math Treasure Hunt
🏴‍☠️ เกม Math Treasure Hunt : ตามล่าหาสมบัติ
🎯 แนวคิดเกม

ผู้เล่นรับบทเป็น นักสำรวจ ที่ต้องออกเดินทางตามหา "หีบสมบัติ" โดยในแต่ละจุดของแผนที่จะมีโจทย์คณิตศาสตร์ซ่อนอยู่ หากตอบถูกจึงจะสามารถเดินทางไปยังด่านถัดไปได้ เมื่อผ่านครบทุกด่านจะได้รับสมบัติและเป็นผู้พิชิตเกาะแห่งคณิตศาสตร์

🎮 วิธีเล่น
ผู้เล่นเริ่มต้นที่ Start
เดินผ่านทั้งหมด 10 ด่าน
แต่ละด่านมีโจทย์สมการ 1 ข้อ
ตอบถูก → ไปด่านต่อไปและรับไอเทม
ตอบผิด → ลองใหม่ หรือหัก 1 คะแนน
เมื่อผ่านครบทุกด่าน จะเปิดหีบสมบัติได้
🗺️ ด่านที่ 1 : ได้กุญแจทอง 🔑

โจทย์จะเป็นสมการง่าย ๆ 


🏆 การให้คะแนน
ตอบถูกครั้งแรก = 10 คะแนน
ตอบถูกครั้งที่สอง = 7 คะแนน
ตอบถูกครั้งที่สาม = 5 คะแนน

คะแนนเต็ม 100 คะแนน

ระดับผู้เล่น
🥉 Explorer : 60–69 คะแนน
🥈 Treasure Hunter : 70–89 คะแนน
🥇 Master Hunter : 90–100 คะแนน
